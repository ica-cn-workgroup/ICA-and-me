# 4. 传感器与执行单元

监测数据的质量与过程控制应用的执行一样重要。在污水处理中，有许多标准化的传感器，如：pH、压力、流量、溶解氧、污泥浓度MLSS，这些传感器的维护要求很低。而其他的传感器，由于需要确保测量质量达到具体的维护要求则需要更深入的分析、维护和检查，如$$NH_4^+$$，$$NO^{-}_3$$和$$PO_4^{3-}$$： 。

在线控制装置需要强调的几方面如下：

* 包括传感器、分析仪和其他测量仪器在内的所有仪表装置都不再是污水系统控制的瓶颈。但是，有一点需要牢记的是，任何时候测量都需要进行质量检验。当然，目前还没有检验在线传感器的通用标准，而且我们也要认识到并重视控制循环依赖于传感器发送的每一个数值。
* 如果没有进行充分的数据质量检验的话，在线测量永远都不能用于控制。因此，仪器永远都需要与充分的数据筛查、测量处理以及或多或少的测量复杂特征提取结合在一起使用。

此外，传动器运行不佳也会摧毁最好的控制方法的结果，这一点会在4.3节中详述。那么，污水处理控制就又存在另一大风险。一个高效运转的系统通常会保证出水水质达标运行。因此，如果一个传感器失效，那么自然就会影响到出水达标。污水厂自动控制会提高效率但是会降低运行的安全性。在所有的运行过程中必须要考虑这点，第12节会深入讨论其中的几方面。

## 4.1 在线传感器

早在上世纪70年代，缺少足够的传感器被认为是污水处理系统自动控制的主要障碍，尤其是现场检测仪表。在第一届ICA会议上，在线仪表开发被给予了很多关注。会议制定了一些共同策略，如：如何进一步开发在线传感器以及如何定义用于自动控制的主要参数。污水处理控制用的传感器虽然与其他工业用传感器看起来一样，但是后者并不能真正适应污水处理的苛刻条件。当时已经认识到，污水厂绩效需要连续评估。但是，像溶解氧、电导率和浊度等变量当时已经可以得到可靠测量，而流量、悬浮物、有机质、氨氮和硝态氮等许多新的产品才刚刚开始出现。有报道称，可商业化应用的氨氮和硝态氮探针可以用于污水厂，但是前提是要保证恒温和常量缓冲，并在合适的时间间隔内增加合适的标样（Briggs, 1973）。

当时，许多仪表产品只是在实验室或中试环境中进行了测试，产品必要的稳定性测试还很缺乏。上世纪70年代中期，只有在线的COD仪表，但是日常维护量缺非常大。而且更重要的是，污水厂运营方对如何使用这些传感器信息只有很模糊的概念，因此很多人最终还是放弃了使用这种分析仪器的想法。很显然，任何一种产品都需要有一个明确的目标。如果运行人员和维护人员无法意识到这点，那么维护传感器校正和正常运行的兴趣就将很快消退。当这些改进的产品已经可以市场化应用的时候，正是这些针对先进仪器应用的初期困难产生了对产品本身的许多质疑，而且导致先进设备的应用被不必要地拖延。

曝气是一项关键的运行参数，因此DO传感器被认为尤其重要。商业化DO传感器早在上世纪80年代就开始作为控制基础进行日常使用。但是，正如Ingildsen等（2002a）和Jeppsson等人（2002）的文章中所提到的，直到2000年初期DO控制仍然没有得到完全使用。而那些用于营养物浓度测量的产品要么是不可商业化应用，要么是认为在实际应用中并不是那么可靠。但是，过去二十年的发展已经发生了变化（表1），而且产品可信度的提升已经得到性能特点明确定义的极大推动，同时产品的标准化测试方法也得到了发展（ISO 15839:2003）。但是，仍然有很多生产商并不愿意去遵循ISO标准（Rieger, 2011）。

除了比较常见的水质测量之外，还有一些可以用于控制的其他产品，如：呼吸计量仪、挥发性脂肪酸VFA传感器和碱度传感器（Vanrolleghem 和 Lee, 2003）。正如Jeppsson等人（2002）提到的，传感器已不再是ICA时间的瓶颈。但是，测量仪器的应用仍然需要重视如何获取可靠信息的问题。有效控制的另一大障碍在于缺乏污水厂灵活性和可控性。

目前，仪表产品种类的标准化已经可以区分、比较和选择最合适的产品，不仅是从技术的角度而且也通过计算购置成本的经济成本角度（表2）。设备本身的投资成本通常在仪器全生命周期成本中只占很小一部分。

Clemson（1974）提出的仪表愿望清单至少花了几十年才变成了现实。营养物分析仪也已经开发出了原位传感器，DO传感器现在已基于发光原理（LDO传感器），而且不再需要膜。上世纪70年代，当时认为污泥层高度应当成为基础的控制参数。当时，还对超声探测器进行了测试。今天，在线污泥层传感器已经十分可靠稳定，并已用于日常应用。

在厌氧污泥消化运行中，沼气产量、沼气质量、挥发性酸成分、温度、进泥速率和碱度已早在上世纪70年代就已得到测量。当时，还认为微生物活性需要进行测量（Bargman等，1973）。而与沼气产量相关的挥发性酸/碱度比例就是这类测量指标之一。有关厌氧处理相关仪表装置的全面回顾文章可以参考Spanjers和van Lier\(2006\)。

与此同时，氧化还原电位ORP也成为微生物状态的测量指标之一。但是，在1973年还没有可靠的在线转换器用于测量挥发性酸、碱度或者ORP。在很多地方有迹象表明，ORP可能是较接近微生物好氧和厌氧活动的主要指标，但是由于测量的不稳定导致基于ORP控制污水厂的尝试最后归于失败。Sekine等人（1985）证明了ORP作为硝化反应指标的新的发展。Wareham等人（1993）则利用ORP作为硝酸盐转折点的指标。Caulet等人（1998）报道了基于ORP脱氮控制的成功应用案例，主要是应用在几座小型活性污泥处理厂。Cecil（2003）在丹麦欧登塞一座污水厂（Ejby Mølle，30万服务人口）利用ORP测量成功控制了反硝化阶段。当还原氧化电位降低到下限值或者当氨氮浓度达到上限值时，曝气就会重新启动。类似地，氨氮测量也被用于控制曝气。当氨氮浓度达到较低的限值，曝气就会停止。相关方法的详细内容请参考Cecil（2008）。

现代化的和自动化的污水处理厂主要依靠在线仪表装置。即使仪表装置今天已经大大改善，但是它们仍然会被污染或者失效。因此，就需要围绕这些问题开展相应的工作并对传感器做出调整。对于测量质量的检查需要给予高度重视。浓度原位传感器（光学和离子选择传感器）需要进行定期检查和调整。深度处理单元，如后置反硝化滤池，有大量仪表和控制回路给予在线仪表进行控制决策，而且需要24小时无人值守。当然，也需要控制安全网以及容错算法和方法学。纠错的时间间隔依赖于测量的媒介种类、传感器特性、测量范围、测量软件以及工艺的测量敏感度。构建一个健全的数据采集系统用于控制和决策是十分关键的。校正和维护需要有经验和富有活力的员工。当然，人工调试需要太多的操作工时间以及提升控制和仪表水平。Lumley等人（2009）和Nivert等人（2009）解决了这一问题，并且在瑞典哥德堡Rya污水厂提出了仪表自调整的方法。这极大地改善了控制质量。

## 4.2 呼吸测量法

1973年呼吸测量法被认为“可能是好氧过程最重要的主要控制手段，主要是因为它可以应用于以下几方面：（1）测量活性污泥的活性，（2）提供有机负荷率F/M用于生物处理控制，（3）提供好氧反应器控制措施，包括如何放置曝气器，（4）能够测量沉淀池效果，（5）测定污染物的可处理性”（William Garber, 参见Bargman等, 1973）。1972年，Robert Arthur报道了一种呼吸测定仪（Arthur, 1972）。Fujimoto等人（1981）提出了一种利用呼吸速率测量来控制反应器DO浓度的方法。随后，一系列测量技术和仪器装置陆续得以开发。但是，有关呼吸原理和各种呼吸测定仪之间存在巨大的困惑。一部分困惑在于仪器的安装位置，而另外的困惑则源于呼吸测定仪与污水厂之间的联系。这一情况迫切需要对呼吸测量原则及其在污水处理控制中的应用进行广泛的研究。Peter Vanrolleghem（Vanrolleghem和Verstraete, 1993; Temmink等人, 1993）和Henri Spanjers（Kapwijk等人，1993；Spanjers等人, 1993, 1994）在呼吸测定方面拥有大量的经验。他们利用不同的呼吸测量原则，并对不同方法进行了系统性比较（Spanjers和Vanrolleghem, 1995）。1993年，我们加入了国际水质协会IAWQ（IWA的前身）的一个工作组，并计划就该主题撰写一份“科学技术报告”。Spanjers等人（1996）报道了相关工作的进展。随后，Spanjers等人（1998a）在最终报告中又报道了呼吸测定仪所有可能的内容及其用于控制的应用。

呼吸速率可以用于测定活性污泥系统的许多方面（Spanjers等人, 1999）。Vanrolleghem等人（1999）描述了利用呼吸测定仪在活性污泥模型中进行参数预测。“呼吸测定仪工作组”出版了第二份“科学技术报告”（Copp等人, 2002），其中对大量控制策略进行了评估，主要利用呼吸测定仪做为前馈和反馈控制的基础。大部分这些机制还只是在模拟中进行了测试。作为早期预警系统，呼吸测定仪是一项具有吸引力的应用，例如：检测上游毒性或者测定进水可生物降解有机底物浓度。利用呼吸测定仪用于毒性保护的生产性应用已用于工业过程。Vanrolleghem等人（1994）报道了一个非常好的案例。另一项利用呼吸测定仪用于医院废水毒性检测的案例参见Vanrolleghem等人（1996a）。

2.4节中曾提到，早期人们曾意识到，整个曝气池可以被用作一个呼吸测定仪。根据DO质量平衡，这看起来是一项很直截了当的工作。进水和出水的DO浓度可以测定，饱和浓度假设为已知。但是，作为空气流量函数的氧传递速率$$K_La$$不是静态的，因此计算呼吸速率就很重要。呼吸速率$$r$$是基于每分钟时间尺度在变化，然而$$K_La$$方程却缓慢变化。两项参数都必须进行评价。只有$$K_La$$已知的情况下呼吸速率才能进行评价，反之亦然。

假设$$K_La$$已知，利用短时间尺度可以计算时间变量r。同样地，正如前面所说，通过鉴定试验可以利用干扰气流计算$$K_La$$。Goto和Andrews（1985）利用DO和气流测量进行了呼吸速率在线预测。整个过程分为两步：第一，预测氧传递速率（$$K_La$$），并据此计算氧气消耗速率。Lindberg和Carlsson（1996a）进一步开发了一种基于复杂筛选算法的呼吸速率评价过程。他们也考虑了当DO传感器动力不能被忽视的情况。在这种评价方案中，利用一种筛选的随机游走模型来模拟随时间变化的呼吸速率值，而非线性函数$$K_La$$则利用一种指数模型进行模拟。一项数值研究说明了这种方法的优势，而且实际数据被应用到软件传感器并取得了满意的结果。

如果在DO闭合回路控制情况下预测呼吸速率，就产生了一个新问题。那么，空气流量就有DO控制器进行测定。这种闭合回路使得识别更加复杂。没有DO控制时，空气流量和DO浓度之间的关系将由DO动态变化规律所决定。而在闭合回路中，同样的数据也会用来确定DO控制器的动力学。Holmberg等人（1988）解决了这一问题，而且显示$$K_La$$和呼吸速率都可以在线计算，同时控制DO浓度。这意味着我们可以在线追踪呼吸速率的发展变化。Carlsson（1993）开发了一种 可以在正常溶解氧闭合回路控制中使用的呼吸速率估计函数。预测过程分为两部分：第一步，基于一组数据预测氧传递速率$$K_La$$的模型，非线性$$K_La$$函数用一种静态限定分段线性模型进行模拟。一旦$$K_La$$确定，呼吸速率就很容易根据DO和控制流量测量进行计算。

尽管上世纪70年代就已经有了极大的期待，基于呼吸测定仪的控制还是没有得到大力发展。虽然有很多控制方案，但是大部分并没有进行过生产型应用。我认为，目前的DO和营养物传感器技术已经满足反馈控制。但是，呼吸测量仪仍然是水体早期预警系统一项可行的方法。

## 4.3 执行单元

早期人们曾认为，驱动器可能会限制控制能力，例如过低最大容量或者过高最低容量。这可能是新型控制策略未能得到大范围接受的主要障碍，许多现存污水处理厂在设计之初并未考虑实时控制。连续调整控制的能力对获得平稳动态控制也至关重要，例如：使用变频传动电力驱动。

在瑞典的一座小型工业污水处理厂，由于DO水平过高，该厂希望能进行DO控制。我们试图弄清楚如何控制DO，然后很快发现尝试任何控制都没有意义。该厂共有3台空气压缩机，我们评估认为只有一台使用，而且DO浓度非常高。平均半台压缩机容量就已经足够。当我待在位于地下的曝气反应器时，出于好奇我让操作工把三台压缩机都打开，结果令我印象深刻。当抱起睡眠快速上升然后很快溢流到地面，我开始跑向地下室的出口。这是我唯一一次在一座污水处理厂中的惊险经历。

一座美国的市政污水厂共有四座压缩机。而0.6台左右压缩机就已满足平均供气量。而在另一座污水厂，由于一级泵的设计DO控制没有任何意义。这种开关泵由于过度设计，以至于每小时只需要开启5分钟就足够泵入进水。因此，每小时这座污水厂都会经受巨大的水力冲击，导致我们很难解决相应的问题。没有DO控制或者分段进水控制系统能够抵消由于泵系统设计不足带来的问题。

我们曾经进行过DO浓度辨识试验。空气流量设定值在控制面板上进行人工更改，但是限定值改变却没有使DO浓度发生任何变化。我们花了一些时间才发现是气流蝶阀被堵。蝶阀通过机械连杆被连接到一个气压传动装置。连杆已经绕到反应器周围的围栏。但是，有人已经在围栏外增加了护网，所以连杆不能再移动。

通常，阀门设计不足也会影响控制。一个问题可能是阀门的非线性行为。另外一个问题是，水厂控制用的阀门运行范围只占它们一小部分范围。这使得控制变得困难而且也不精确。Olsson和Newell（1999, 第24章）有详细描述。在一次污水厂试验中，我们曾尝试利用大型进水阀门测量污水水平来控制进水流量速率。阀门的运行范围大约为1米，但是有趣的是，在0.2米的工作点周围，阀门运行范围被限定为0.02米。换句话说，控制行为非常不精确而且困难。
